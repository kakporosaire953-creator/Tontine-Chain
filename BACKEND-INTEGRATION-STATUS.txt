ENDPOINTS API UTILISES:
========================

Auth:
- POST /auth/request-otp
- POST /auth/verify-otp

User:
- GET /users/me
- PATCH /users/me
- GET /users/me/score
- GET /users/me/balance
- GET /users/me/payouts

Groups:
- GET /groups
- POST /groups
- GET /groups/{id}
- POST /groups/{id}/join
- GET /groups/{id}/stats

Contributions:
- GET /contributions/pending
- POST /contributions/{id}/pay
