# TontineChain

TontineChain est une application permettant de securiser les tontines au Benin en utilisant la technologie blockchain.

## Le Probleme
Les tontines traditionnelles font face a de nombreux risques :
- La Cale Seche : les collecteurs disparaissent avec les fonds.
- Les Carnets Falsifies : perte ou modification des donnees.
- La vulnerabilite des participants.

## La Solution
TontineChain utilise des smart contracts pour automatiser et securiser la collecte et la distribution des fonds. 
Aucune intervention humaine ne peut modifier le code une fois deployee.

## Fonctionnalites
- Inscription et authentification securisee.
- Tableau de bord de suivi des tontines.
- Messagerie integree.
- Assistant virtuel (YAO).

## Equipe
Projet developpe dans le cadre du MIABE Hackathon.
