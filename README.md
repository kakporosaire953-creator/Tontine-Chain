# 💎 TontineChain - Frontend Premium
**Interface Utilisateur Elite | Hackathon MIABE 2026**

Ce dépôt contient le client React de la plateforme **TontineChain**. Il est conçu pour offrir une expérience fluide, sécurisée et inclusive pour la gestion des tontines numériques.

---

## 🎨 Design & Expérience
- **Elite Dark Mode** : Une esthétique sobre et luxueuse pour rassurer les utilisateurs sur le caractère sérieux et sécurisé de la plateforme.
- **Animations Framer Motion** : Transitions fluides entre les vues pour une sensation d'application native.
- **Accessibilité Audio** : Boutons d'assistance vocale en langue Fon intégrés pour l'inclusion financière.

---

## 🛠️ Stack Technique
- **React 19**
- **Vite** (HMR ultra-rapide)
- **Tailwind CSS 4.0**
- **Lucide React** (Iconographie)
- **Axios** (Communication API)

---

## 🚀 Lancement Local

1. **Installation** :
   ```bash
   npm install
   ```

2. **Développement** :
   ```bash
   npm run dev
   ```

3. **Build Production** :
   ```bash
   npm run build
   ```

---

## 📱 Fonctionnalités Clés
- **Tableau de Bord Dynamique** : Vue d'ensemble des cotisations, gains espérés et score de confiance.
- **Gestion de Groupe** : Détails des cycles, enchères de décotes et votes de permutation.
- **Profil KYC** : Interface de vérification d'identité sécurisée.
- **Notifications** : Centre d'alertes en temps réel pour ne rater aucune échéance.
