## 🎨 Palette de Couleurs (Style TontiGo)

### Fonds
```css
--bg-primary: #0a0a0a      /* Noir profond (comme TontiGo) */
--bg-secondary: #141414    /* Noir légèrement plus clair */
--bg-tertiary: #1f1f1f     /* Gris très sombre */
```

### Textes
```css
--text-primary: #ffffff    /* Blanc pur */
--text-secondary: #d4d4d4  /* Gris très clair */
--text-tertiary: #a3a3a3   /* Gris moyen */
```

### Bordures
```css
--border-color: #2a2a2a    /* Gris très sombre */
--border-hover: #3a3a3a    /* Gris sombre */
```

### Accents Or/Jaune (comme TontiGo)
```css
--accent-gold: #FFB81C         /* Or TontineChain */
--accent-gold-hover: #ffc94d   /* Or plus clair */
```

## 🌟 Effets Visuels Spéciaux

```css
  radial-gradient(circle at 20% 50%, rgba(255, 184, 28, 0.03) 0%, transparent 50%),
  radial-gradient(circle at 80% 80%, rgba(22, 163, 74, 0.03) 0%, transparent 50%);
```

### Lueur sur Logo
```css
filter: drop-shadow(0 0 10px rgba(255, 184, 28, 0.3));
```

### Lueur sur Brand "Tontine"

```css
text-shadow: 0 0 20px rgba(255, 184, 28, 0.3);
```
