## 🆘 BESOIN D'AIDE?

### Problèmes courants

**Images floues après compression**
→ Voir OPTIMISATION_IMAGES.md section "Troubleshooting"

**CSS cassé après minification**
→ Vérifier la syntaxe, utiliser version non-minifiée

**Attributs ARIA ne fonctionnent pas**
→ Tester avec WAVE ou lecteur d'écran

**Meta tags ne s'affichent pas**
→ Vider cache, tester avec Facebook Debugger
