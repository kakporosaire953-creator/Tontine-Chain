# 🌙 Mode Sombre avec Accents Verts - TontineChain

## ✨ Palette Noir/Blanc/Vert/Or

J'ai ajouté des accents verts stratégiques en mode sombre pour créer une palette cohérente avec l'identité TontineChain.

---

## 🎨 Palette de Couleurs Complète

### Fonds
```css
--bg-primary: #0a0a0a      /* Noir profond */
--bg-secondary: #141414    /* Noir légèrement plus clair */
--bg-tertiary: #1f1f1f     /* Gris très sombre */
```

### Textes
```css
--text-primary: #ffffff    /* Blanc pur */
--text-secondary: #d4d4d4  /* Gris très clair */
--text-tertiary: #a3a3a3   /* Gris moyen */
```

### Accents Or (Stats, Badges)
```css
--accent-gold: #FFB81C         /* Or principal */
--accent-gold-hover: #ffc94d   /* Or plus clair */
```

### Accents Verts (Actions, Liens, Success)
```css
--accent-green: #16a34a        /* Vert principal */
--accent-green-light: #22c55e  /* Vert clair */
--accent-green-bright: #4ade80 /* Vert lumineux */
```

## 📝 Éléments avec Accents Verts

### Navigation
```css
/* Liens hover et active en vert */
.nav-menu a:hover → Vert lumineux
.nav-menu a.active → Vert lumineux
```

### Sidebar
```css
/* Liens hover et active en vert */
.sidebar-menu a:hover → Vert lumineux + fond gris
.sidebar-menu a.active → Vert lumineux + bordure gauche verte
```

### Sous-titres
```css
/* Descriptions en vert clair */
.hero-subtitle → Vert clair
.section-header p → Vert clair
.step-description → Vert clair
.dashboard-header p → Vert clair
```

### Labels
```css
/* Labels de formulaires en vert clair */
.stat-label → Vert clair
.stat-box-label → Vert clair
label → Vert clair
```

### Textes Importants
```css
/* Textes mis en avant en vert lumineux */
.hero-tag → Vert lumineux
.trust-badge span → Vert lumineux
.info-box p → Vert lumineux
.why-benefit span → Vert lumineux
```

### Icônes de Succès
```css
/* Icônes check en vert lumineux */
.fa-check → Vert lumineux
.fa-check-circle → Vert lumineux
.fa-shield-check → Vert lumineux
.trust-item i → Vert lumineux
```

### Liens
```css
/* Liens hover en vert lumineux */
a:hover → Vert lumineux
```

### Boutons
```css
/* Boutons secondaires en vert */
.btn-secondary → Bordure + texte vert clair
.btn-secondary:hover → Fond vert + texte noir

/* Boutons primaires gardent le vert */
.btn-primary → Vert avec lueur
.btn-green → Vert avec lueur
```

### Inputs
```css
/* Focus en vert */
input:focus → Bordure vert clair + lueur verte
textarea:focus → Bordure vert clair + lueur verte
select:focus → Bordure vert clair + lueur verte

/* Input valide en vert */
input:valid → Bordure verte
```

### Progress Bars
```css
/* Fill en gradient vert */
.progress-fill → Gradient vert + lueur verte
```

### Status Indicators
```css
/* Payé en vert */
.member-status.paid → Fond vert transparent + texte vert lumineux

/* En attente en or */
.member-status.pending → Fond or transparent + texte or
```

### Montants et Prix
```css
/* Montants en vert lumineux */
.price → Vert lumineux
.amount → Vert lumineux
.tontine-stat-value → Vert lumineux
```
